Then, we I meet Vundle

I want this, I want that

Bundle "garbas/vim-snipmate"

...
Bang! The magic :BundleInstall " install

:BundleInstall! " update
Meet my friends!
ctrlp.vim

Full path fuzzy file, buffer, mru, tag, ... finder for Vim.

  • ctrl+p: trigger ctrlp.vim
  • enter: open
  • ctrl+t: open in new tab
  • f5: purge cache
  • ctrl+y: create new file
ack.vim

ack is a tool like grep, optimized for programmers

  • :Ack xxx: search whatever you want
  • o, go: open/preview
  • t, T: open with new tab
  • q: close
vim-snipmate.vim

concise vim script that implements some of TextMate's snippets features in Vim

  • tab: just tab and enjoy the fun.
vim-fugitive

a Git wrapper so awesome, it should be illegal

  • basic commands: :Gwrite, :Gread, :Gremove, :Gmove
  • view status: :Gstatus, :Gdiff
  • dive into history: :Gedit, :Glog, :Ggrep
  • three way merge: Gdiff, :diffget //2(//3)
